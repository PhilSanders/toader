# Toader Game 

Built with Phaser game framework - [Phaser.io](www.http:phaser.io)

### Setup

You may need to install Gulp and required plugins with 'sodo'

`$ sudo npm install gulp gulp-webserver gulp-useref`

### Run

Run the web server

`$ gulp webserver`

### Build

Build distribution 

`$ gulp build`
